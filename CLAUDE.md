这是华南农业大学人工智能协会关于睿抗机器人培训会的OpenCV入门和简单图片处理培训项目。

## 项目结构

```
CV培训/

├── docs/      # 教程文档目录

└── README.md  # 项目说明
```

## 开发说明

- 这是一个OpenCV计算机视觉培训项目
- 项目当前处于初始化阶段，培训和教程内容待完善
- 主要面向OpenCV初学者，涵盖基础图像处理操作
- 每次更改需要上传git并提交PR，确保代码质量和项目结构清晰
- 代码应遵循PEP8规范，注释清晰，易于理解

## Python环境

使用OpenCV进行培训，通常需要安装：
- opencv-python
- numpy

典型的安装命令：
```bash
pip install opencv-python numpy
```
