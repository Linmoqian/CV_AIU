### noisy/ - 含噪图片

| 文件名 | 用途 | 生成方式 |
|--------|------|----------|
| noisy-gaussian.jpg | 高斯滤波对比 | 添加高斯噪声 |

| clean-reference.jpg | 对比参考 | 干净的原图 |

**使用Python添加噪声示例：**
```python
import cv2
import numpy as np

# 添加高斯噪声
img = cv2.imread('clean.jpg')
noise = np.random.normal(0, 30, img.shape).astype(np.uint8)
noisy = cv2.add(img, noise)
cv2.imwrite('noisy.jpg', noisy)

# 添加椒盐噪声
sp = img.copy()
# 白点
coords = [np.random.randint(0, i, int(0.05*img.size)) for i in img.shape]
sp[coords[0], coords[1], coords[2]] = 255
# 黑点
coords = [np.random.randint(0, i, int(0.05*img.size)) for i in img.shape]
sp[coords[0], coords[1], coords[2]] = 0
cv2.imwrite('salt-pepper.jpg', sp)
```

## 测试视频建议

### 需要的视频类型

1. **color-tracking.mp4**
   - 内容：移动的彩色物体（如红色球）
   - 用途：颜色追踪练习
   - 时长：10-30秒

2. **car-moving.mp4**
   - 内容：行驶的车辆
   - 用途：运动检测
   - 时长：10-30秒

### 获取方式

**自己录制：**
```python
import cv2

# 打开摄像头
cap = cv2.VideoCapture(0)

# 定义编码器
fourcc = cv2.VideoWriter_fourcc(*'mp4v')
out = cv2.VideoWriter('output.mp4', fourcc, 20.0, (640, 480))

while cap.isOpened():
    ret, frame = cap.read()
    if ret:
        out.write(frame)
        cv2.imshow('Recording', frame)

        if cv2.waitKey(1) & 0xFF == ord('q'):
            break
    else:
        break

cap.release()
out.release()
cv2.destroyAllWindows()
```
