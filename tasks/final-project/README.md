## 项目A：智能文档扫描器

### 项目描述
实现一个自动矫正文档透视畸变的程序，输入一张拍摄的文档照片，输出矫正并增强的扫描效果。

### 功能要求

#### 基础功能（60分）
1. **图像输入**（10分）
   - 支持读取图像文件

2. **边缘检测**（15分）
   - 使用Canny边缘检测
   - 正确检测文档边缘

3. **轮廓查找与筛选**（15分）
   - 找到文档的四个角点
   - 使用多边形近似

4. **透视变换**（20分）
   - 计算透视变换矩阵
   - 应用变换矫正图像

#### 进阶功能（30分）
1. **图像增强**（15分）
   - 自适应阈值二值化
   - 对比度增强

2. **形态学优化**（15分）
   - 去除噪声
   - 连接断开的字符

#### 额外功能（10分）
- GUI界面
- 批量处理
- 参数可调

### 提交内容
- 源代码：`scanner.py`
- 测试图片及结果
- 技术报告（说明实现思路）

## 项目B：颜色识别分类器

### 项目描述
实现一个能够识别图像中不同颜色物体并计数的程序。

### 功能要求

#### 基础功能（60分）
1. **多颜色识别**（30分）
   - 支持至少3种颜色（红、绿、蓝）
   - 使用HSV色彩空间

2. **物体计数**（30分）
   - 统计每种颜色的物体数量
   - 过滤噪声（面积阈值）

#### 进阶功能（30分）
1. **可视化标注**（15分）
   - 绘制边界框
   - 显示颜色标签和数量

2. **位置信息**（15分）
   - 输出每个物体的中心坐标
   - 在图像上标注

#### 额外功能（10分）
- 支持自定义颜色
- 实时摄像头检测
- 生成统计报告

### 提交内容
- 源代码：`color_classifier.py`
- 测试图片及结果
- 技术报告

## 项目C：简易美颜相机

### 项目描述
实现一个实时摄像头美颜滤镜，包括磨皮、美白等功能。

### 功能要求

#### 基础功能（60分）
1. **摄像头读取**（15分）
   - 打开摄像头
   - 实时显示视频流

2. **磨皮效果**（25分）
   - 使用双边滤波
   - 可调节强度

3. **美白效果**（20分）
   - Gamma校正或色彩调整
   - 可调节强度

#### 进阶功能（30分）
1. **多滤镜切换**（15分）
   - 至少3种滤镜效果
   - 键盘快捷键切换

2. **实时参数调整**（15分）
   - 使用滑动条调整参数
   - 实时预览效果

#### 额外功能（10分）
- 拍照保存功能
- 滤镜叠加
- 其他创意效果

### 提交内容
- 源代码：`beauty_camera.py`
- 演示视频或截图
- 技术报告

## 常见问题

**Q: 可以使用深度学习吗？**
A: 本次培训专注于传统图像处理，不建议使用深度学习方法。

**Q: 可以使用其他库吗？**
A: 核心功能必须使用OpenCV，辅助功能可以使用其他库（如numpy、matplotlib等）。

**Q: 遇到问题怎么办？**
A:
1. 查阅教程文档
2. 在群里提问
3. 参加答疑时间

**Q: 可以参考网上的代码吗？**
A: 可以参考思路，但必须自己实现，并在报告中注明参考来源。
